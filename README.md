# SOC Master Engine

A Python-based SOC automation engine that simulates:

- Brute force attack detection
- Password spraying detection
- Web attack detection (SQLi / Phishing)
- Incident merging
- JSON structured reporting
- Automated PDF report generation

## Project Phases

Phase 1 – Log Parsing & Detection  
Phase 2 – JSON Conversion & Severity Summary  
Phase 3 – Automated Professional PDF Reporting  

Built as part of hands-on SOC analyst training.
